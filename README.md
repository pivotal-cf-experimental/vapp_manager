# VappManager

A gem which provides ruby classes and executables for installing and configuring a vApp in vCloud Director.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'vapp_manager'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vapp_manager
